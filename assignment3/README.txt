3.1  -- 'wc.py'
How to use wc.py:

1.  source the wc.sh file
2.  in the terminal run wc with least 1 argument.
    Any argument provided should the path of a file, or filename in the current dir.
    wc will print out the word count any file provided as an argument.
    To print out the word count of all files in the directory use wc *.


3.2 -- 'test_complex.py'
